Astralens

A real-time GPU-accelerated black hole simulation built with Python and Taichi.

## Overview

Astralens is a real-time visualization of a black hole and its accretion disk, created during a graphics programming hackathon. The project combines scientific inspiration with cinematic rendering techniques while maintaining real-time performance.

This project was built using AI-assisted development and served as my introduction to GPU programming, graphics programming, rendering pipelines, and version control with Git. Rather than following a tutorial, I approached the project by learning through building and iterating.

## Features

- Real-time black hole visualization
- Temperature-based accretion disk rendering
- Cinematic photon ring
- Procedural nebula background
- Procedural starfield and distant galaxies
- GPU acceleration using Taichi
- Cinematic opening camera
- Color grading and visual enhancements
- Real-time performance (~55–60 FPS on my hardware)

## Technologies Used

- Python 3.12
- Taichi
- Git
- GitHub

## How to Run

```bash
python sim.py
```
